## Installation

Install [Docker](https://www.docker.com/).

Copy and edit `.env.example`

```bash
cp .env.example .env
nano .env
```

Create folders and `database.ini`

```bash
source .env
bash setup.sh
```

## Usage

Start Omeka S

```bash
docker-compose up -d
```
